# Parkline Apartments | Coming Soon

A one-page Coming Soon website for Parkline Apartments, a 362-residence purpose-built
rental community at 4500 Park Street in downtown Niagara Falls, developed in
partnership between Elite Developments and the City of Niagara Falls.

## Structure

- `index.html` : complete site (HTML, CSS and JS in one file)
- `assets/` : optimized renderings, groundbreaking photography, logos, favicons and social share image
- `.nojekyll` : ensures GitHub Pages serves files as-is
- `robots.txt` : allows indexing

## Deploy on GitHub Pages

1. Create a new repository and upload the contents of this folder to the repository root.
2. In the repository, open Settings, then Pages.
3. Under Build and deployment, set Source to "Deploy from a branch", choose the `main` branch and the `/ (root)` folder, then save.
4. The site will publish at `https://<username>.github.io/<repository>/`.

## Before going live

- In `index.html`, update `og:image` and add an `og:url` and canonical link using the final absolute domain so social previews resolve correctly.
- All contact links are live: tel 1.844.484.4184 and sales@elitedevelopments.com.

## Notes

- Fonts load from Google Fonts (Archivo) with system fallbacks.
- Reduced-motion preferences are respected; all animation is disabled for those visitors.
- All renderings and photography are the supplied project assets, resized and compressed for the web. Logos are used exactly as provided.
